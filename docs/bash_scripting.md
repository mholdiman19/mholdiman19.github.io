## Episode 1

* `which $SHELL` => `/bin/bash`
* `echo "Hi Mom"` => `Hi Mom`
* To automate the above command, open a text editor, such as `nano`.
* `nano himom.sh`
* Create script

```bash

#!/bin/bash
echo "Hi Mom"

```
* save script in nano
* do `ls` to confirm your file is saved
